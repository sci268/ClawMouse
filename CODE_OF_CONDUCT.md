# 行为准则

为了让 ClawMouse 社区保持开放、可协作和可持续，参与讨论、提 Issue、提 Pull Request 时请遵循下面的原则。

## 我们鼓励

- 尊重不同技术背景和使用场景
- 用事实、复现步骤和代码证据讨论问题
- 对新想法保持开放，同时说明约束和风险
- 在指出问题时给出可执行建议

## 我们不欢迎

- 人身攻击、羞辱、嘲讽
- 持续性骚扰
- 恶意刷屏、挑衅或故意制造冲突
- 公开泄露他人的隐私、密钥或敏感环境信息

## 协作方式

- Bug 报告尽量附复现步骤
- 功能请求尽量说明目标场景
- 大改动优先先讨论设计方向
- 如果改动影响文档或示例，请同步更新

## 维护者处理方式

维护者可以对不符合上述原则的内容进行编辑、隐藏、锁定或拒绝合并。

如果你认为某次讨论已经偏离技术合作目标，请停止升级冲突，转而聚焦复现、证据和最小修改方案。

# 安全说明

感谢你帮助改进 ClawMouse 的安全性。

这个项目具备真实桌面控制能力，包括：

- 鼠标与键盘输入
- 窗口聚焦与窗口内点击
- 文本输入与聊天发送
- 截图与窗口区域捕获
- 文件桥接与任务委托

因此请把它视为具有较高操作权限的自动化软件。

## 适合报告的问题

- 可以绕过预期限制的桌面控制问题
- 会导致敏感数据泄露的截图、日志、桥接问题
- 会导致任意执行或越权操作的 MCP 接口问题
- 默认配置中不安全的行为

## 不建议公开提交的内容

请不要在公开 Issue 中直接附带以下信息：

- 本机绝对路径
- 账号、Cookie、Token、API Key
- 含有敏感窗口内容的截图
- 可直接复现你个人环境隐私的桥接任务文件

## 报告方式

如果问题涉及真实安全风险，建议先通过私下方式联系维护者，而不是直接公开披露。

如果暂时没有专用安全邮箱，至少请做到：

- 先给出风险概述
- 隐去敏感数据
- 提供最小复现步骤
- 说明影响范围和触发条件

## 使用建议

- 不要在高权限桌面会话中长期暴露不受控的 MCP 服务
- 对外提供 HTTP 传输前先限制访问范围
- 桥接目录建议放在可控的本地位置
- 公开仓库前确认 `.gitignore` 已排除运行期产物

## 当前重点关注

- 桥接队列残留文件导致的异常恢复
- OCR / 截图链路中的敏感信息外泄风险
- 聊天发送与窗口识别误操作
- MCP 高层委托接口的边界校验
